# AI Agent Instruction Templates

## Basic Requests

### Simple Task
```
Please [action] [object/topic].
```
Example: *Please summarize this document.*

### Task with Context
```
I need help with [task]. [Context/background information].
```
Example: *I need help with data analysis. I'm working on a quarterly sales report.*

### Task with Specific Output
```
Create a [type of output] that [specific requirement].
```
Example: *Create a report that highlights key trends and actionable insights.*

## Analysis & Research

### Data Analysis
```
Analyze [data/document] and identify:
- [Aspect 1]
- [Aspect 2]
- [Aspect 3]
```

### Comparative Analysis
```
Compare [A] and [B] in terms of:
- [Criteria 1]
- [Criteria 2]
- [Criteria 3]

Provide a recommendation based on your analysis.
```

### Research Request
```
Research [topic] and provide:
1. An overview of [specific aspect]
2. Current trends or recent developments
3. Key challenges and opportunities
```

### Investigation
```
Investigate [issue/topic] and explain:
- What is happening
- Why it's happening
- Potential implications
- Recommended next steps
```

## Creation & Generation

### Content Creation
```
Write a [type of content] about [topic] that:
- Is [length/style requirement]
- Targets [audience]
- Includes [specific elements]
```

### Code Generation
```
Create [type of code/script] that:
- [Primary function]
- [Additional requirement 1]
- [Additional requirement 2]

Use [specific language/framework] and include comments.
```

### Design/Plan Creation
```
Develop a [plan/strategy/framework] for [objective] that includes:
- [Component 1]
- [Component 2]
- [Component 3]
- Timeline and milestones
```

## Modification & Refinement

### Improvement Request
```
Improve [content/code/document] by:
- [Improvement area 1]
- [Improvement area 2]
- [Improvement area 3]

Keep [elements to preserve].
```

### Rewrite Request
```
Rewrite [content] to be more [characteristic]:
- Target audience: [audience]
- Tone: [tone]
- Length: [length requirement]
```

### Optimization
```
Optimize [code/process/content] for [goal]:
- Priority: [main priority]
- Constraints: [any limitations]
- Maintain [essential features]
```

## Explanation & Summarization

### Explanation Request
```
Explain [concept/topic] as if I'm [level of expertise].
Include:
- Key concepts
- Practical examples
- Common misconceptions
```

### Step-by-Step Explanation
```
Explain how to [task/process] with:
- Clear step-by-step instructions
- Prerequisites needed
- Common pitfalls to avoid
- Expected outcomes
```

### Summary Request
```
Summarize [content] focusing on:
- Main points
- Key takeaways
- Actionable insights

Keep it under [length].
```

## Problem-Solving

### Troubleshooting
```
I'm experiencing [problem] when [context].
- What I've tried: [attempted solutions]
- Error message: [if applicable]
- Expected behavior: [what should happen]

Help me identify the cause and solution.
```

### Decision Support
```
I need to decide between [Option A] and [Option B] for [purpose].
Consider:
- [Factor 1]
- [Factor 2]
- [Factor 3]

Provide a recommendation with reasoning.
```

## Specific Format Requests

### Structured Output
```
Provide your response in the following format:

**[Section 1 Title]**
[Content requirements]

**[Section 2 Title]**
[Content requirements]

**[Section 3 Title]**
[Content requirements]
```

### Table Format
```
Present [information] in a table with columns:
- [Column 1]
- [Column 2]
- [Column 3]
```

### List Format
```
List [items/options/ideas] ranked by [criteria].
For each item include:
- Brief description
- Pros and cons
- [Additional relevant info]
```

## Multi-Step Tasks

### Sequential Task
```
Complete the following tasks in order:

1. First, [task 1]
2. Then, [task 2]
3. Next, [task 3]
4. Finally, [task 4]

Confirm completion of each step before moving to the next.
```

### Iterative Task
```
[Task description]

After completing, ask me for feedback and iterate until I'm satisfied.
Key requirements:
- [Requirement 1]
- [Requirement 2]
```

## Advanced Patterns

### Role-Based Prompting
```
Act as a [role/expert] and [task].
Consider:
- [Relevant perspective 1]
- [Relevant perspective 2]
- [Relevant perspective 3]
```

### Constraint-Based
```
[Task description] with the following constraints:
- MUST: [Required elements]
- MUST NOT: [Forbidden elements]
- SHOULD: [Preferred elements]
- Length: [specification]
```

### Output Validation
```
[Task description]

Before providing your final answer:
1. Verify [criteria 1]
2. Check [criteria 2]
3. Ensure [criteria 3]
```

## Quick Phrases

### Clarification
- "Can you elaborate on [aspect]?"
- "What do you mean by [term/phrase]?"
- "Could you provide more details about [topic]?"

### Refinement
- "Make it more [adjective]"
- "Simplify this explanation"
- "Add more technical details"
- "Focus specifically on [aspect]"

### Continuation
- "Continue with [next part]"
- "Expand on [specific point]"
- "Now address [next topic]"

### Confirmation
- "Is this correct: [statement]?"
- "Verify that [condition]"
- "Double-check [element]"
